A JSON Schema validator implementation in Java which aims for correctness and performance, in that order

Sample usage

  • create a {@link org.eel.kitchen.jsonschema.schema.JsonSchemaFactory};
  • create a {@link org.eel.kitchen.jsonschema.schema.JsonSchema} using that factory;
  • for each instance you wish to validate:
    • create a {@link org.eel.kitchen.jsonschema.main.ValidationContext};
    • validate using the created schema.

NOTE: the API is very likely to change in the future! It will be stabilized once version 0.5.0 is reached.
